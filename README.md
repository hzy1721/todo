# `Microsoft To Do` web version

## Tech stack

- framework: Vue 3 + VueUse + Pinia
- language: SCSS + TS
- persistance: Dexie (IndexedDB) + localStorage
- svg icons: [iconfont](https://www.iconfont.cn/)
